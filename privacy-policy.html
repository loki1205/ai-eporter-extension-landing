Privacy Policy for Page Exporter

Page Exporter is designed with privacy as a core principle. All webpage processing happens locally in your browser. No exported content is transmitted, stored, or shared externally.

Data Collection

Page Exporter does not collect, store, sell, or share personal data.

The extension temporarily accesses webpage content only to generate exports requested directly by the user.

How the Extension Works

The extension reads visible webpage content locally in the browser and converts it into:

  • TXT files
  • Markdown files
  • DOCX documents

No webpage content leaves the user's device.

Permissions Used

  • activeTab — used to access the currently active webpage after explicit user interaction.
  • scripting — used to extract visible webpage content for formatting and export generation.

Third-Party Services

Page Exporter does not use:

  • analytics services
  • tracking scripts
  • advertising networks
  • external APIs
  • remote code execution
